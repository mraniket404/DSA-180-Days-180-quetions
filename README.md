# 🚀 DSA 180 Days Challenge (Java)

Welcome to my **DSA 180 Days Challenge** repository!

This repository documents my **180-day journey** of solving **Data Structures and Algorithms (DSA)** problems using **Java**. Every day, I solve one problem, explore multiple approaches, analyze its complexity, and maintain well-organized code along with detailed PDF explanations.

The primary goal of this challenge is to improve problem-solving skills, master DSA concepts, and prepare for coding interviews at top product-based companies.

---

# 📌 About This Challenge

Learning Data Structures and Algorithms is not just about solving problems—it's about understanding different ways to solve the same problem efficiently.

For every problem, I try to implement multiple approaches whenever possible, starting from the basic solution and gradually moving towards the most optimized one.

Each day's solution includes:

- 📝 Problem Statement
- 💡 Multiple Approaches
- 🧠 Logic & Explanation
- ☕ Java Solution
- 📄 PDF Notes
- ⏱️ Time Complexity
- 💾 Space Complexity

Whenever applicable, the approaches include:

- Brute Force
- Better Approach
- Optimal Approach
- Mathematical Solution
- Iterative Solution
- Recursive Solution
- String-Based Solution
- Hashing
- Two Pointers
- Sliding Window
- Dynamic Programming
